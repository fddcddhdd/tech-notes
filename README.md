# tech-notes

技術的な疑問をHTMLノートとして蓄積するためのシンプルな静的サイトです。

## 構成

```text
tech-notes/
├── index.html
├── styles/
│   └── site.css
├── notes/
│   └── sample.html
├── AGENTS.md
└── README.md
```

## 使い方

1. GitHubにこの中身をアップロードする
2. GitHub Pagesを有効化する
3. `notes/` 配下にHTMLノートを追加する
4. `index.html` にリンクを追加する

## Codexへの依頼例

```text
質問:
LaravelのRequestで、$request->f_option と $request['f_option'] の違いを説明する技術ノートを作ってください。

条件:
- notes/ 配下に新しいHTMLを作る
- index.html にリンクを追加する
- 初心者にも分かるように説明する
- よくある勘違いも入れる
- 参照URLを末尾に入れる
```
